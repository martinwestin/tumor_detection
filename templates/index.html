Classify CT Scans of the Brain

This is a website where you can classify CT scan images of the brain, by letting a Convolutional Neural Network decide if there is a tumor. If there turns out to be a tumor in the brain the image is depicting, the model will try to classify the tumor as one of the following types:

  • Meningioma
  • Glioma
  • Pituitary
